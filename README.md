# 🚀 Aprendiendo React

Este repositorio contiene todos mis proyectos y prácticas mientras aprendo **React**.  
La idea es ir construyendo, probando y entendiendo conceptos paso a paso, aplicándolos en pequeños proyectos.

---

## 📚 Fuente de aprendizaje

Estoy basando mi aprendizaje principalmente en:

- 📺 Playlist de YouTube (curso principal)
- 💻 Repo base del curso de midudev

A partir de eso, adapto, practico y creo mis propias versiones de cada ejercicio o proyecto.

---

## 📂 Estructura del repositorio

Cada carpeta representa una práctica o mini-proyecto:

- 00-hola-mundo → Primer contacto con React
- 01-practica → useState, useEffect
- 03-ta-te-ti → Juego de TaTeTi
- 04-prueba-tecnica-react → Ejercicio tipo prueba técnica
- 05-react-buscador-peliculas → Buscador con debounce
- 06-shopping-cart → Carrito de compras

---

## 🧠 Qué estoy aprendiendo

A lo largo de este repo estoy practicando:

- ⚛️ Componentes en React
- 🔄 Hooks (`useState`, `useEffect`)
- 📦 Manejo de estado
- 🔍 Búsquedas y debounce
- 🧩 Lógica de aplicaciones reales
- 🧪 Simulación de pruebas técnicas
- 🛒 Estructura de apps más complejas (como shopping cart)
